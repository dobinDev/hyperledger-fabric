1. Type: 이 리소스가 AWS::Serverless::Function 리소스 유형을 사용하여 정의된 AWS Lambda 함수임을 나타냅니다.
2. CodeUri: 함수 코드의 위치를 지정합니다.
3. AutoPublishAlias: AWS Lambda가 함수의 새 버전을 자동으로 게시할 때 사용할 별칭을 지정합니다.
4. Handler: 람다 핸들러 클래스를 지정합니다.
5. Events: 람다 함수를 트리거하는 이벤트를 지정합니다.
    5.1 Type: 이것이 Api 이벤트 소스임을 지정합니다.
    5.2 Properties: API 이벤트의 경우 API Gateway가 응답해야 하는 HTTP 메서드 및 경로를 정의합니다.
